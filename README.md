# AWS Quiz KR — Frontend

Next.js 기반 AWS 자격증 문제집, 퀴즈 웹 앱입니다. 백엔드 REST API와 통신해 로그인 상태, 문제 집계, 약점 분석, 추천, 오늘의 문제 같은 학습 기능을 화면에 붙입니다.

## 기술 스택

- Next.js, React, TypeScript
- Tailwind CSS
- Axios

---

## 메인, 약점 분석, 추천 UX

백엔드의 임베딩, 약점 벡터 로직은 서버에 두고, 프론트는 아래처럼 화면과 진입 경로로 노출합니다.

### 홈(`home-client`)

- 공지, 문제집 목록, 검색, 페이지네이션, 정답률, 이벤트 안내 등 메인 랜딩.
- 로그인 시 `/auth/me/weakness-comment` 로 AI 약점 코멘트와 진행 정보를 받아, 최근 풀이 50문제 채움 여부에 따라 문구, 버튼 활성화를 바꿉니다.
- 비로그인 사용자에게는 약점, 추천 진입을 로그인 유도 문구로 처리합니다.

### 오늘의 문제

- 홈에서 오늘의 문제 는 `Quiz` 페이지에 `?daily=true` 로 이동합니다.
- API는 `/auth/me/daily-questions` 이며, 목표 자격증이 없으면 백엔드 정책에 맞춰 안내, 모달로 처리합니다.

### AI 추천 약점 문제 모드

- 홈의 약점 분석 영역에서 준비가 된 사용자는 `/Quiz?recommended=weakness` 로 들어갑니다.
- 문항 목록은 `/auth/me/recommended-questions` 로 받으며, 화면에 추천 이유(`recommendReason`) 를 표시합니다.

### 퀴즈 공통

- 문제집 ID 기반 일반 풀이와 위 특수 모드를 같은 `Quiz` 페이지에서 분기합니다.
- 이전, 다음, 난이도, 유형, 힌트 토글, 제출 확인 모달, 채점 결과(정오답, 오답 바로가기)를 제공합니다.

### 관리자: 카테고리 AI 추천

- 문제집 관리 화면에서 작성 중인 문항에 대해 `POST /admin/question-categories/recommend` 를 호출하는 AI 추천 버튼으로, 임베딩 기반 카테고리 후보를 받아 폼에 반영합니다.

### SEO, 라우트 메타

- 공개 페이지별 `layout`과 `lib/seo` 등으로 메타, canonical, robots 를 라우트 단위로 조정합니다.

## 핵심 기능 (요약)

- 회원가입, 로그인, 비밀번호 찾기, 재설정, 이메일 인증 코드 UX
- 마이페이지: 목표 자격증, 계정 요약, 비밀번호 변경
- 메인: 공지, 문제집 검색, 페이지네이션, 이벤트 안내
- 퀴즈: 일반, 오늘의 문제, AI 추천 약점 모드, 채점 결과
- 관리자 UI: 공지, 문제집, 문항 CRUD, AI 카테고리 추천, 통계 대시보드
- 헤더 인증 상태, SEO 개선
